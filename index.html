üìù Setup Instructions

  1. Get your API keys:
    • GROQ API Key (Required): Get it from GROQ Console
    • Google (Gemini) API Key (Optional): Get it from Google AI Studio
  2. Enter your API keys in the form below
  3. Click "Generate Config" to create your configuration file
  4. Click "Download LUMA" to get the application package
  5. Extract the downloaded LUMA.zip file
  6. Move the generated build_config.py file into the extracted folder
  7. Run LUMA.exe to start your voice assistant!
Note: Your API keys are processed locally in your browser. They are never sent to any server.
Important: Windows Defender or your antivirus might flag LUMA.exe as suspicious because it's an unsigned executable. This is normal for Python applications built with PyInstaller. The code is open source and safe - you can verify the source code on GitHub and build it yourself. If you prefer, you can:
  • Use Windows Defender to allow the app
  • Build from source using the instructions in the README
  • Run the Python script directly instead of using the executable
